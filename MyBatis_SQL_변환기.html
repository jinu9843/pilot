MyBatis SQL 변환기 (DBeaver용)

스프링부트 콘솔의 Preparing: + Parameters: 로그를 붙여넣으면 ? 자리에 파라미터를 순서대로 넣은 실행 가능한 SQL을 만들어 줍니다.
HTML + JS 2개 파일을 같은 폴더에 두고 HTML을 더블클릭하세요.

① MyBatis 로그 전체 붙여넣기

② 쿼리 (Preparing)

③ 파라미터 (Parameters)

④ DBeaver 실행용 SQL

⑤ 파라미터 치환 미리보기

순서 원본 타입 SQL 값
변환 후 표시됩니다.
사용법
1. 콘솔에서 Preparing: / Parameters: 로그 복사
2. ①에 붙여넣기 → 로그에서 분리
3. SQL 변환 클릭
4. ④ 결과를 DBeaver에 붙여넣고 실행

치환 규칙
· String/Date/Timestamp → '값'
· Integer/Long/Double → 숫자 그대로
· null → NULL
