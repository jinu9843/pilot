쿼리 로그 테이블 뷰어

AS-IS 서버 쿼리 로그(scm_family_cd | tech_cd | ...)를 붙여넣으면 엑셀처럼 보기 편한 표로 보여줍니다.
HTML + JS 2개 파일을 cursor 폴더에 두고 HTML을 더블클릭하세요.

① 쿼리 로그 붙여넣기

② 결과 표

로그를 붙여넣고 표로 보기를 눌러주세요.

사용법
1. 이클립스/서버 로그에서 헤더 + 데이터 행 복사
2. 붙여넣기 → 표로 보기 (단축키: Ctrl+Enter)
3. 컬럼 많으면 가로 스크롤, 행 많으면 세로 스크롤
4. 엑셀로내려면 엑셀용 복사(TSV) → 엑셀 붙여넣기

지원 형식
· 구분자: | 또는 탭
· 1행 = 컬럼명, 2행부터 = 데이터
· 빈 값 | | 도 정상 인식
