Java Debug → Swagger JSON 변환기

AS-IS Java 디버그 출력(sort_cnt ={0})을 Swagger Request Body용 JSON("sortCnt": "0")으로 변환합니다.
snake_case → camelCase, {<null>} → null, {} → 빈 문자열 처리를 지원합니다.

입력 (Java 디버그 값 붙여넣기)

출력 (Swagger JSON)

변환 미리보기

원본 키 변환 키 (camelCase) 원본 값 JSON 값
변환 후 표시됩니다.

입력 예시

sort_cnt ={0} pri_user_id={} sale_typ={} rnd_unit={1} period_from={202601} period_to={202606}
