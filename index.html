Nexacro -> Spring Boot 백엔드 생성기

넥사크로 요청값, 응답값, 넥스코어 서비스 메서드, 기존 SQL XML을 붙여넣으면 오프라인에서 Spring Boot 초안 코드를 생성합니다. 결과로 RequestDto, QueryDto, ResponseDto, Service, ServiceImpl, Mapper, Mapper.xml, Controller, 서비스 소스 분석, 서비스 변환 초안, Swagger용 요청 예시 JSON이 만들어집니다.

입력 형식은 너무 엄격하지 않게 만들었습니다. SQL XML 변환 시 아래 규칙을 적용합니다.
1. <isNotEqual property="STD_WEEK" compareValue="Y"> → <if test="stdWeek != 'Y'"> (작은따옴표 test + !="Y" 조합은 오류 → 큰따옴표 test + != 'Y' 사용)
2. NVL(M$arrYmList[].ym$_MO_R,0) → NVL(M${item.ym}_MO_R,0) + <foreach collection="arrYmList" item="item">
3. $DYN_SQL_PIVOT$ → ${dynSqlPivot} (camelCase)
